# 3D-RPG
- 鼠标左键点击控制移动、战斗，右键点击并拖动控制镜头移动，滑动中键控制视野缩放
- 攻击石头能将其击飞反击石头人
- 敌人能够随机巡逻或站桩守卫，玩家进入视野后追击玩家，玩家超出视野后等待一段时间脱战，并返回原始巡逻点
- 靠近传送门按E键传送，游戏过程中按S存档，按L读档
- 击杀敌人获得经验值，经验值满则会升级
